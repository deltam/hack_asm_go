# hack_asm_go

Assembler for HackComputer(nand2tetris.org) written in Go

## Usage

```
# generate ./sample.hack
go run /cmd/Assembler/Assembler.go _asm/sample.asm
# output STDOUT
go run /cmd/Assembler/Assembler.go -s _asm/sample.asm
```
